# Together AI (together-ai)

Together AI is an AI acceleration cloud delivering fast, scalable, and reliable generative-AI infrastructure. The Together API serves open-source and proprietary foundation models for chat, embeddings, vision, audio, image and video generation, fine-tuning, code execution, and dedicated GPU compute.

## APIs

| API | Description |
|---|---|
| Together Chat Completions API | OpenAI-compatible chat across hundreds of open-source models. |
| Together Completions API | Legacy text completions endpoint. |
| Together Embeddings API | Dense vector embeddings (e5, BGE) for retrieval/RAG. |
| Together Rerank API | Cross-encoder reranking of candidate passages. |
| Together Images API | Text-to-image (FLUX, Nano Banana Pro, Stable Diffusion). |
| Together Video API | Text/image-to-video generation across model tiers. |
| Together Audio API | TTS (MiniMax, Cartesia, Kokoro, Orpheus) and STT (Whisper, Parakeet). |
| Together Vision API | Multimodal vision/document understanding. |
| Together Fine-Tuning API | Managed LoRA, full SFT, and DPO across the catalog. |
| Together Files API | Upload datasets and batch input files. |
| Together Models API | Lists hundreds of models with metadata and pricing. |
| Together Batch API | Async inference at up to 50% off serverless rates. |
| Together Code Interpreter API | Sandboxed Python execution alongside model calls. |
| Together Evaluations API | LLM-as-judge evaluations and reports. |
| Together Dedicated Endpoints API | Single-tenant GPU-backed endpoints, hourly billing. |
